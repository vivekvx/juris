# M7.2 Sidebar Screenshots

Screenshots require an authenticated browser session. 
Run `npm run dev` in /frontend and log in to capture.

## What changed

**Before:**
- Active nav: bg-secondary (neutral gray), rounded-lg (12px)
- Sidebar width: 240px expanded, 56px collapsed
- No loading state in conversations list
- No empty state in conversations list
- No conversations list on mobile
- "Collapse" with text + icon

**After:**
- Active nav: bg-primary/10 (accent blue tint), rounded-md (8px)
- Sidebar width: 220px expanded, 52px collapsed (tighter)
- Loading: 4 skeleton rows with shimmer animation
- Empty: ChatCircle icon + "No conversations yet" label
- Mobile: full conversation list in sheet
- Collapse: icon-only when collapsed, icon+label when expanded
- All items: `focus-visible:ring-2 focus-visible:ring-ring`
- NavLink extracted: one component for all nav items
- Footer: Settings unified with same NavLink pattern
- Brand: 18px Scales + semibold 14px, py-3 (was py-4)
- Section label: uses text-label utility (uppercase, tracked)
- Transition: duration-200 (was 150)
